Pergunta 1: O que é um algoritmo?

Um conjunto de instruções para resolver um problema

Pergunta 2: Qual é a finalidade de um loop em programação?

Para declarar variáveis
Para executar uma sequência de instruções várias vezes

Pergunta 3: O que é uma variável em programação?

Uma condição lógica
Um tipo de operador
Um espaço de memória para armazenar valores

Pergunta 4: Qual é a diferença entre == e === em JavaScript?

Não há diferença, ambos são iguais
== compara apenas os valores, enquanto === compara os valores e os tipos de dados
== compara apenas os tipos de dados, enquanto === compara os valores

Pergunta 5: O que é um array?

Um tipo de variável que armazena apenas números inteiros
Uma estrutura de dados que armazena uma coleção de elementos
Um tipo de loop

Pergunta 6: O que é uma função em programação?

Um tipo de variável
Um bloco de código reutilizável que realiza uma tarefa específica
Um tipo de estrutura condicional

Pergunta 7: O que é um operador lógico?

Um tipo de variável que armazena valores booleanos
Um tipo de função em JavaScript
Um tipo de símbolo que realiza operações lógicas em valores booleanos

Pergunta 8: O que é um algoritmo de ordenação?

Um algoritmo que converte dados em diferentes tipos de dados
Um algoritmo que organiza elementos em uma sequência específica
Um algoritmo que verifica a validade de uma expressão lógica

Pergunta 9: O que é recursividade em programação?

Um tipo de loop
Um conceito onde uma função pode chamar a si mesma
Um tipo de variável

Pergunta 10: O que é um pseudocódigo?

Um tipo de código JavaScript
Um tipo de algoritmo de ordenação
Uma descrição de alto nível de um algoritmo que utiliza uma linguagem simples e não específica de programação

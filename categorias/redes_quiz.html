Quiz


Pergunta 1: O que é um protocolo de rede?

Um tipo de cabo utilizado para conectar dispositivos
Um conjunto de regras que governa a comunicação entre dispositivos em uma rede
Um tipo de roteador

Pergunta 2: Qual é a função do DNS (Sistema de Nomes de Domínio) na Internet?

Gerenciar o tráfego de rede
Converter nomes de domínio em endereços IP
Fornecer segurança para redes sem fio

Pergunta 3: O que é um endereço IP?

Um número único atribuído a cada dispositivo conectado a uma rede
Um tipo de protocolo de segurança
Um tipo de software de rede

Pergunta 4: O que é um roteador?

Um dispositivo que fornece acesso à Internet
Um dispositivo que direciona o tráfego de rede entre diferentes dispositivos
Um tipo de firewall

Pergunta 5: Qual é a diferença entre uma rede LAN e uma rede WAN?

Uma rede LAN é sem fio, enquanto uma rede WAN é com fio
Uma rede LAN é local e limitada a uma área geográfica pequena, enquanto uma rede WAN abrange uma área geográfica maior e pode conectar várias LANs
Não há diferença, ambos os termos se referem ao mesmo tipo de rede

Pergunta 6: O que significa TCP/IP?

Transmissão de Controle de Pacotes/Internet Protocol
Transmissão de Conteúdo e Protocolo de Internet
Transmissão de Comunicação Privada/Protocolo de Interface

Pergunta 7: O que é um ataque DDoS?

Um tipo de ataque cibernético que desativa completamente uma rede
Um ataque que visa roubar informações pessoais dos usuários
Um ataque que sobrecarrega um servidor com um grande volume de tráfego de rede

Pergunta 8: O que é NAT (Tradução de Endereço de Rede)?

Uma técnica de criptografia usada para proteger dados transmitidos pela rede
Um tipo de protocolo de roteamento usado em redes sem fio
Um método que permite que dispositivos em uma rede privada usem um único endereço IP público para se comunicar com a Internet

Pergunta 9: O que é um switch de rede?

Um dispositivo que conecta dispositivos em uma rede sem fio
Um dispositivo que filtra e encaminha pacotes de dados para seus destinos na rede
Um tipo de firewall que protege contra ataques de negação de serviço

Pergunta 10: O que é um cabo de fibra óptica?

Um tipo de cabo usado para conectar dispositivos a uma rede sem fio
Um cabo que usa sinais elétricos para transmitir dados entre dispositivos em uma rede
Um cabo que usa pulsos de luz para transmitir dados em alta velocidade
